# Worktree Management Skill

## Purpose

Provides patterns for managing Git worktrees to enable parallel development across multiple feature branches without context switching.

## When to Use

- When setting up parallel development environments
- When managing multiple feature branches simultaneously
- When coordinating work across worktrees
- When cleaning up completed worktrees

## Key Features

- Worktree creation and configuration
- Parallel branch management
- Worktree coordination patterns
- Cleanup and removal procedures
